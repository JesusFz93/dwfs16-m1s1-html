Existen varios tipos de listas

Listas desordenadas

Las listas desordenadas son aquellas que no tienen un orden especifico, por ejemplo, una lista de ingredientes para una receta.

  • Harina
  • Huevos
  • Leche
  • Sal

Listas ordenadas

Las listas ordenadas son aquellas que tienen un orden especifico, por ejemplo, una lista de pasos para realizar una tarea.

  1. Preparar los ingredientes
  2. Mezclar los ingredientes
  3. Calentar la mezcla
  4. Disfrutar

Listas anidadas

Las listas anidadas son aquellas que contienen otras listas, por ejemplo, una lista de ingredientes para una receta que a su vez contiene una lista de pasos para preparar cada ingrediente.

  • Harina
  • Huevos
  • Leche
  • Sal
    1. Mezclar los ingredientes
    2. Calentar la mezcla
    3. Disfrutar

Listas de definicion

Las listas de definicion son aquellas que contienen una lista de terminos y una lista de definiciones, por ejemplo, un diccionario.

HTML
Es un lenguaje de marcado que se utiliza para el desarrollo de paginas web
CSS
Es un lenguaje de estilos que se utiliza para dar estilo a las paginas web
JavaScript
Es un lenguaje de programacion que se utiliza para dar interactividad a las paginas web
